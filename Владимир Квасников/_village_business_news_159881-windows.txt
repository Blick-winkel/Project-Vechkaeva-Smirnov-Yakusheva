С 13 января 2015 года компания Microsoft прекратит базовую поддержку операционной системы Windows 7. Об этом сообщает ИТАР-ТАСС.После этого в течение пяти лет будет действовать расширенная поддержка операционной системы. Она включает в себя бесплатные обновления безопасности, критические для системы, и платные дополнения, касающиеся функциональности.Также в режим расширенной поддержки перейдут и некоторые другие продукты компании: Windows Phone 7.8, Windows Server 2008, Windows Storage Server 2008, Exchange Server 2010 и Dynamics.В апреле Microsoft уже прекратила поддержку Windows XP. После этого несколько наиболее крупных клиентов договорились с корпорацией о платной подписке на обновления. В противном случае под угрозой оказались бы компьютеры госслужащих.
