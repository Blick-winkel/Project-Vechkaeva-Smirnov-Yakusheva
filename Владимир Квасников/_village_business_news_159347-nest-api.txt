Производитель термостатов Nest Labs, недавно приобретённый Google, запустил специальную программу для сторонних разработчиков. Теперь они смогут использовать API для интеграции своих сервисов в экосистему «умного» дома.Так, например, температурой в доме теперь можно будет управлять при помощи голосовых команд Google на Android. Также термостат можно будет подключить к сервису IFTTT, который позволяет создавать разнообразные последовательности реакций на то или иное событие: например, можно будет составить алгоритм, согласно которому пользователю сразу же должно прийти сообщение на мобильный, если в его доме сработал детектор дыма. С другими примерами использования API можно ознакомиться здесь.Первые инвесторы Nest Labs — Google Ventures и Kleiner Perkins Caulfield and Byers — уже заявили о своей готовности поддержать стартапы, которые направлены на развитие технологий для «умного» дома за счёт использования API Nest.
