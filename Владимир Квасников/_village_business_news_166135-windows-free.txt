Следующая версия операционной системы Windows может оказаться бесплатной. По крайней мере, для тех пользователей, которые уже пользуются Windows 8. Об этом сообщает техноблог BGR со ссылкой на заявление главы индонезийского подразделения Microsoft Андреса Дианторо.До этого компания никогда не предлагала бесплатных обновлений до следующей версии своей операционной системы. Например, за переход на Windows 8.1 в России нужно заплатить как минимум 6 тысяч рублей. Делая новую версию Windows бесплатной, Microsoft подстраивается под поведение пользователей, которые уже привыкли получать обновления для мобильных ОС без какой-либо оплаты.Корпорация должна представить новую версию Windows уже сегодня, 30 сентября. Её точное название пока что не раскрывается. Среди вариантов — Windows 9, Windows Threshold или Windows TH. В системе должна вновь появиться кнопка «Пуск», поддержка голосового помощника Cortana и возможность запускать несколько виртуальных рабочих столов.
